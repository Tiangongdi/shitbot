# 文档编写指南 -- 该文档告诉如何给ShitBot编写文档,如果要编写文档请看这个文档

## 内容介绍
该文档告诉如何给ShitBot编写文档，包括格式，内容等。
## 文档格式
- 文档采用 Markdown 格式编写。
- 每个文档都有一个介绍，标题层级为 `#` 开始，格式通常为`# 文档标题 -- 文档介绍 `，例如 `# 文档编写指南 -- 该文档告诉如何给ShitBot编写文档`。
- 文档内容按照功能模块进行组织，每个模块都有一个标题层级为 `##` 开始 ，例如 `## 内容介绍`，然后这个后面就是模块的内容。
## 文档建议
- 文档内容要简洁明了，避免使用复杂的词汇。
- 文档内容要按照功能模块进行组织，每个模块都有一个标题层级为 `##` 开始 ，例如 `## 内容介绍`，然后这个后面就是模块的内容。
- 工具类型的文档一般有下面3个模块
    - 工具介绍
    - 工具参数
    - 最佳实践
