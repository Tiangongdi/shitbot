# 搜索工具使用指南 -- 本文档详细介绍了 ShitBot 工具模块中搜索相关工具的使用方法、参数说明、使用示例以及最佳实践。

## 工具概述

搜索工具模块包含两个核心工具：`search_web`（网络搜索）和 `webbot_task`（浏览器任务执行）。这两个工具配合使用，可以完成从搜索关键词到获取详细网页内容的完整流程。

## search_web - 网络搜索

### 工具介绍
`search_web` 工具用于在网络上搜索信息，支持自定义搜索关键词和返回结果数量。

### 工具参数
- `query`（必需）：搜索查询词，字符串类型
- `count`（可选）：返回结果数量，整数类型，默认值为 5

### 使用示例
```
搜索最新的 AI 新闻：
search_web(query="AI 最新新闻", count=10)

搜索 Python 教程：
search_web(query="Python 教程 入门")
```

### 最佳实践
- 使用具体的关键词可以获得更准确的搜索结果
- 对于需要大量信息的场景，可以适当增加 count 参数
- 搜索结果可以配合 webbot_task 使用，获取详细内容

## webbot_task - 浏览器任务执行

### 工具介绍
`webbot_task` 工具用于执行浏览器操作任务，包括查看网页信息、点击网页、填写表单等浏览器修改功能。这是一个功能强大的浏览器自动化工具。

### 工具参数
- `query`（必需）：告诉WebBot要执行的任务，字符串类型

### 使用示例
```
打开百度并搜索"人工智能"：
webbot_task(query="打开百度并搜索人工智能")

访问新闻网站并提取内容：
webbot_task(query="访问 https://news.example.com 并提取最新的科技新闻")

填写表单：
webbot_task(query="打开登录页面并填写用户名和密码")
```

### 最佳实践
- query 参数应该明确描述需要执行的任务
- 可以执行复杂的浏览器操作流程
- 支持网页导航、内容提取、表单填写等多种操作
- 确保提供的 URL 是有效的且可访问的

## 典型使用场景

### 场景一：研究某个主题
1. 使用 search_web 搜索相关关键词
2. 从搜索结果中选择有价值的网页
3. 使用 webbot_task 获取详细内容

### 场景二：获取最新资讯
1. 使用 search_web 搜索最新新闻
2. 使用 webbot_task 提取新闻详情
3. 整合多个来源的信息

### 场景三：自动化网页操作
1. 使用 webbot_task 打开网页
2. 执行点击、填写表单等操作
3. 提取操作结果

## 常见问题

**Q: 搜索结果不准确怎么办？**
A: 尝试使用更具体的关键词，或者调整搜索词的表述方式。

**Q: webbot_task 无法执行任务怎么办？**
A: 检查任务描述是否清晰，确保网页可以正常访问，尝试将复杂任务拆分为多个简单任务。

**Q: webbot_task 和之前的 browse_page 有什么区别？**
A: `webbot_task` 是更强大的浏览器自动化工具，不仅可以浏览网页，还可以执行点击、填写表单等交互操作。

## 注意事项

- 搜索和浏览功能需要网络连接
- 部分网站可能有反爬虫机制，可能无法正常提取内容
- 请遵守网站的使用条款和版权规定
- 浏览器自动化操作需要确保 Playwright 已正确安装
