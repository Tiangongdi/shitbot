# 搜索工具使用指南 -- 本文档详细介绍了 ShitBot 工具模块中搜索相关工具的使用方法、参数说明、使用示例以及最佳实践。

## 工具概述

搜索工具模块包含两个核心工具：`search_web`（网络搜索）和 `browse_page`（网页浏览）。这两个工具配合使用，可以完成从搜索关键词到获取详细网页内容的完整流程。

## search_web - 网络搜索

### 工具介绍
`search_web` 工具用于在网络上搜索信息，支持自定义搜索关键词和返回结果数量。

### 工具参数
- `query`（必需）：搜索查询词，字符串类型
- `count`（可选）：返回结果数量，整数类型，默认值为 5

### 使用示例
```
搜索最新的 AI 新闻：
search_web(query="AI 最新新闻", count=10)

搜索 Python 教程：
search_web(query="Python 教程 入门")
```

### 最佳实践
- 使用具体的关键词可以获得更准确的搜索结果
- 对于需要大量信息的场景，可以适当增加 count 参数
- 搜索结果可以配合 browse_page 使用，获取详细内容

## browse_page - 网页浏览

### 工具介绍
`browse_page` 工具用于打开指定的网页并提取内容，支持指定要提取的内容类型。

### 工具参数
- `url`（必需）：要访问的网页 URL，字符串类型
- `query`（必需）：告诉网站提取总结助手要在这个网站里面提取和总结的内容，字符串类型

### 使用示例
```
浏览百度百科页面并提取人物信息：
browse_page(url="https://baike.baidu.com/item/人工智能", query="人工智能的定义和发展历史")

浏览新闻网站并提取新闻内容：
browse_page(url="https://news.example.com/article/123", query="新闻的主要内容")
```

### 最佳实践
- query 参数应该明确指定需要提取的内容类型
- 对于复杂的网页，可以多次调用 browse_page 提取不同部分的内容
- 确保提供的 URL 是有效的且可访问的

## 典型使用场景

### 场景一：研究某个主题
1. 使用 search_web 搜索相关关键词
2. 从搜索结果中选择有价值的网页
3. 使用 browse_page 获取详细内容

### 场景二：获取最新资讯
1. 使用 search_web 搜索最新新闻
2. 使用 browse_page 提取新闻详情
3. 整合多个来源的信息

## 常见问题

**Q: 搜索结果不准确怎么办？**
A: 尝试使用更具体的关键词，或者调整搜索词的表述方式。

**Q: browse_page 无法提取内容怎么办？**
A: 检查 URL 是否正确，确保网页可以正常访问，调整 query 参数使其更明确。

## 注意事项

- 搜索和浏览功能需要网络连接
- 部分网站可能有反爬虫机制，可能无法正常提取内容
- 请遵守网站的使用条款和版权规定
