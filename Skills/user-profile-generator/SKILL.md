# User Profile Generator

通过5个精心设计的问题，快速生成用户画像介绍。

## 核心原则

1. **问题简洁** - 每个问题不超过一句话，避免过于细节
2. **一次一问** - 逐个提问，不要一次性抛出所有问题
3. **自然对话** - 用轻松友好的语气提问
4. **灵活调整** - 根据用户回答的详细程度适当追问或跳过

## 五个核心问题

### 问题1：身份探索
```
"你目前的主要身份是什么？（比如：学生、职场人、自由职业者等）"
```
目的：了解用户的基本身份定位

### 问题2：兴趣爱好
```
"工作/学习之余，你最喜欢做什么？"
```
目的：探索用户的兴趣领域和休闲方式

### 问题3：工作风格
```
"你觉得自己是急性子还是慢性子？做事喜欢提前规划还是随性而为？"
```
目的：了解用户的工作效率和行事风格

### 问题4：人生愿景
```
"如果用一句话描述你理想中的生活状态，会是什么样？"
```
目的：捕捉用户的人生目标和价值观

### 问题5：性格画像（根据前面的回答生成几个符合的MBTI 类型）
```
"你可能是INFJ 或者ENFJ ，对吗？你可能是一个非常有组织的人，也可能是一个非常独立的人。你可能是一个非常有目标的人，也可能是一个非常有自由的人。"
```
目的：获取用户的性格特征描述

## 执行流程

1. **开场白** - 简单介绍目的，开始第一个问题
2. **逐个提问** - 使用 `input` 工具，一次只问一个问题
3. **记录回答** - 将每个回答记录下来
4. **生成画像** - 收集完5个回答后，整合生成完整的用户介绍

## 生成格式

收集完所有回答后，按以下格式生成用户画像：

```markdown
# 用户画像

## 👤 身份定位
[基于问题1的回答，描述用户身份]

## 🎯 兴趣爱好
[基于问题2的回答，总结用户兴趣]

## ⚡ 工作风格
[基于问题3的回答，分析工作效率和风格]

## 🌟 人生目标
[基于问题4的回答，提炼人生愿景]

## 💫 性格特点
[基于问题5的回答，刻画用户的性格特征]

---

## 📝 综合介绍
[用一段流畅的文字，将以上五个方面融合成一段完整的个人介绍]
```

## 注意事项

- 如果用户回答过于简短，可以适当追问一句
- 如果用户回答已经很详细，不要追问，直接进入下一题
- 保持轻松友好的对话氛围
- 最终生成的介绍要自然流畅，避免生硬拼接
