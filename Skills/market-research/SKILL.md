# 市场调研分析技能

本技能提供系统化的市场调研方法论和工具，帮助全面分析项目在当前市场中的定位和前景。

## 核心功能

### 1. 市场规模评估
- TAM/SAM/SOM分析法
- 市场增长率计算
- 细分市场规模估算
- 市场饱和度评估

### 2. 竞争对手分析
- 竞品识别与分类
- 竞争格局矩阵
- 竞品功能对比
- 竞争策略分析

### 3. 行业趋势研究
- PESTEL宏观环境分析
- 技术发展趋势
- 用户需求演变
- 政策法规影响

### 4. SWOT分析
- 优势(Strengths)识别
- 劣势(Weaknesses)评估
- 机会(Opportunities)挖掘
- 威胁(Threats)预警

### 5. 用户研究
- 目标用户画像
- 用户需求分析
- 用户行为研究
- 用户痛点识别

## 调研流程

### 第一步：明确调研目标
1. 确定调研的核心问题
2. 定义调研范围和边界
3. 设定可量化的调研指标

### 第二步：信息收集
使用以下工具收集市场信息：
- `search_web`: 搜索行业报告、市场数据
- `browse_page`: 深入阅读专业报告和分析文章

**推荐信息源：**
- 行业研究报告（艾瑞、易观、IDC等）
- 政府统计数据
- 企业财报和招股书
- 专业媒体和咨询机构

### 第三步：数据分析
1. 整理收集到的数据
2. 进行定量和定性分析
3. 识别关键洞察和模式

### 第四步：生成报告
输出结构化的市场调研报告，包括：
- 执行摘要
- 市场概况
- 竞争分析
- 趋势预测
- 风险评估
- 行动建议

## 使用示例

**示例1：市场规模评估**
```
用户：帮我分析一下在线教育市场的规模
流程：
1. 搜索最新的在线教育市场报告
2. 收集TAM/SAM/SOM数据
3. 分析增长率和趋势
4. 输出市场规模评估报告
```

**示例2：竞争对手分析**
```
用户：分析一下短视频领域的竞争对手
流程：
1. 识别主要竞争对手
2. 收集竞品信息（功能、用户、营收等）
3. 构建竞争矩阵
4. 分析竞争策略
5. 输出竞争分析报告
```

## 注意事项

1. **数据时效性**：优先使用最近1-2年的数据，标注数据来源和时间
2. **数据可靠性**：交叉验证多个信息源，提高数据可信度
3. **客观性**：避免主观偏见，基于事实和数据进行分析
4. **全面性**：从多个维度分析，避免片面结论
5. **可操作性**：提供具体、可执行的建议
